# 获取站点内容列表API

获取整个站点的内容列表，使用`GET`发起请求，请求地址为`/api/v1/contents/{siteId}`

```http
GET /api/v1/contents/{siteId} HTTP/1.1
```

## 请求 URI

| 参数名 | 位置  | 类型   | 必填   | 说明                                          |
| ------ | ----- | ------ | ------ | --------------------------------------------- |
| siteId | path  | 整数   | **是** | 站点Id                                        |
| apiKey | query | 字符串 | 否     | API 密钥，请参考[身份认证](authentication.md) |

## 返回

| 名称             | 类型                                        | 说明     |
| ---------------- | ------------------------------------------- | -------- |
| 200 OK           | [content](/contents/README?id=content) 列表 | 内容列表 |
| 401 Unauthorized | [Error](/error?id=error)                    | 认证错误 |

## 示例

获取指定站点的所有内容列表

### 请求示例

```http
GET /api/v1/contents/1325 HTTP/1.1
X-SS-API-KEY: 7cd22002-27a7-4c5d-ba4d-a1c108a20eaf
```
