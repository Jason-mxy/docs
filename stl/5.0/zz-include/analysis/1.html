显示浏览量

<stl:analysis
	addNum="添加数目"
	channelIndex="栏目索引"
	channelName="栏目名称"
	isAverage="是否显示日均量"
	isDynamic="是否动态显示"
	scope="统计范围"
	since="时间段"
	style="显示样式"
	type="统计类型"
></stl:analysis>

显示页面的浏览量及点击次数、点击率。

属性列表

下表显示所有属于此标签的属性列表。

属性 可能的取值 默认值 简介
channelIndex     栏目索引
channelName     栏目名称
type     显示类型
  PageView 显示访问量（PV）
  UniqueVisitor   显示独立访客量（UV）
  CurrentVisitor   显示当前人数
scope     统计范围
  Page 统计页面
  Site   统计全站
isAverage     是否显示日均量
  true   显示日均量
  false 显示全部
style     显示图标类型
  Number 采用文字显示数字
  Style1 显示第一种图片代表的数字
  Style2   显示第二种图片代表的数字
  Style3   显示第三种图片代表的数字
  Style4   显示第四种图片代表的数字
  Style5   显示第五种图片代表的数字
  Style6   显示第六种图片代表的数字
  Style7   显示第七种图片代表的数字
  Style8   显示第八种图片代表的数字
  Style9   显示第九种图片代表的数字
addNum     显示时在实际数字基础上增加的数字
since     时间段，格式为"数字+日期符号"，日期符号有"y"、"m"、"d"及"h"，分别代表年、月、天及小时。
isDynamic     是否动态显示
  true   动态显示
  false 静态显示

注意

标签 <stl:analysis> 用于显示浏览量，可以将此标签单独使用，也可以放在列表标签内使用。

<stl:analysis> 默认显示生成页面时对应的浏览量，不会显示实时浏览量。如果需要显示实时浏览量数据，需要设置isDynamic属性为true。

属性 type 用于设置显示浏览量的类型，scope 用于设置浏览量的范围，isAverage 用于设置是否显示每日的平均浏览量。

属性 style 用于设置显示的样式，默认直接显示数字，可以设置采用图片显示数字：设置 style="Style..."。

PV(访问量)：即Page View, 即页面浏览量或点击量，用户每次刷新即被计算一次。
UV(独立访客)：即Unique Visitor,访问您网站的一台电脑客户端为一个访客。00:00-24:00内相同的客户端只被计算一次。

属性 since 用于指定时间段，系统将只显示这段时间内的统计数字。since 属性格式为"数字+日期符号"，如"1y"、"3m"、"10d"或"16h"，分别代表一年、三个月、十天及十六个小时。

如果希望实时显示统计数据，应该设置属性 isDynamic 为"True"。

要使 <stl:analysis> 显示浏览量，系统必须将统计设置开启（功能管理 /流量统计管理 /修改统计设置）。

<stl:analysis>标签对应的实体为{stl:analysis}。
