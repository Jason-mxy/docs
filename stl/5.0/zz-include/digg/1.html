掘客

<stl:digg
	badText="不赞同文字"
	goodText="赞同文字"
	isDynamic="是否动态显示"
	isNumber="仅显示结果数字"
	theme="主题样式"
	type="类型"
></stl:digg>

实现掘客功能，可以设置为赞同/不赞同、投个鲜花/扔个鸡蛋、顶一下/踩一下等。

属性列表

下表显示所有属于此标签的属性列表。

属性 可能的取值 默认值 简介
type     类型
  Good   仅显示赞同
  Bad   仅显示不赞同
  All 显示赞同及不赞同
goodText   顶一下 赞同显示文字
badText   踩一下 不赞同显示文字
theme     样式主题
  Style1 样式一
  Style2   样式二
  Style3   样式三
  Style4   样式四
isNumber     是否单独显示数值
  true   显示
  false 不显示
isDynamic     是否动态显示
  true   动态显示
  false 静态显示

注意

type 属性设置显示掘客功能的类型，goodText 及 badText 属性设置显示文字。

theme 属性设置显示样式。

<stl:digg>标签对应的实体为{stl:digg}。
