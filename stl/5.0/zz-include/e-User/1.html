User 用户实体用于获取用户信息，可将实体放到 HTML 属性中。

属性列表

下表显示所有属于此实体的属性列表。

实体项目 简介 使用说明
UserName 账号 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
DisplayName 姓名 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
Email 电子邮箱 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
Mobile 手机号码 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
CreateDate 注册时间 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
LastActivityDate 最近一次活动时间 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
CreateIPAddress 注册 IP 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
Group 用户组 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
Type 用户类型 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
AvatarLarge 头像（大图） 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
AvatarMiddle 头像（中图） 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
AvatarSmall 头像（小图） 必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用
login.submit 登录提交按钮事件 {user.login.submit}只能在<stl:user type="login">中使用,放在登录表单的onsubmit事件中
login.redirectToRegister 跳转到用户注册页面 {user.login.redirectToRegister}只能在<stl:user type="login">中使用,放在按钮或链接的onclick事件中
login.redirectToForgetPassword 跳转到找回密码页面 {user.login.redirectToForgetPassword}只能在<stl:user type="login">中使用,放在按钮或链接的onclick事件中
register.submit 用户注册提交 {user.register.submit}只能在<stl:user type="register">中试用,放在注册表单的onsubmit事件中
forget.submit 用户找回密码提交 {user.forget.submit}只能在<stl:user type="forget">中试用,放在找回密码表单的onsubmit事件中

注意

大部分属性必须在<stl:dynamic>或者<stl:visible type="IsUser">标签内使用。当用户登录时显示对应用户的相关属性，无用户登录时显示为空值。

实体写法为：{User.abc} ，其中 abc 为指定的实体项目，实体不区分大小写。
