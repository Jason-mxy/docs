下拉菜单 

<stl:menu
	channelIndex="栏目索引"
	channelName="栏目名称"
	childMenuDisplay="二级菜单显示方式"
	childMenuHeight="二级菜单高度"
	childMenuWidth="二级菜单宽度"
	groupChannel="指定显示的栏目组"
	groupChannelNot="指定不显示的栏目组"
	isDynamic="是否动态显示"
	isShowChildren="是否显示二级菜单"
	menuHeight="菜单高度"
	menuWidth="菜单宽度"
	styleName="样式名称"
	target="打开窗口目标"
	xPosition="菜单水平位置"
	yPosition="菜单垂直位置"
></stl:menu>

显示栏目下拉菜单，一般用作站点导航栏。

属性列表

下表显示所有属于此标签的属性列表。

属性 可能的取值 默认值 简介
styleName     菜单样式名称
channelIndex     栏目索引
channelName     栏目名称
groupChannel     指定显示的栏目组
groupChannelNot     指定不显示的栏目组
isShowChildren     是否显示二级菜单。
  true   显示二级菜单
  false 不显示二级菜单
menuWidth     菜单宽度
menuHeight     菜单高度
xPosition     菜单水平位置
yPosition     菜单垂直位置
childMenuDisplay     二级菜单显示方式
childMenuWidth     二级菜单宽度
childMenuHeight     二级菜单高度
target     打开窗口目标
isDynamic     是否动态显示
  true   动态显示
  false 静态显示

注意

可以把任何 STL 标签或 HTML 标签作为 stl:menu 标签的子标签，当鼠标移动到 stl:menu 标签的子标签上时将显示菜单。

如果设置了 channelIndex 属性，系统将寻找对应此栏目索引的栏目并显示此栏目的菜单。
如果设置了 channelName 属性，系统将寻找对应此栏目名称的栏目并显示此栏目的菜单。
如果设置了 groupChannel 或者 groupChannelNot 属性，系统将根据栏目组别寻找菜单进行显示。

<stl:menu>标签对应的实体为{stl:menu}。
