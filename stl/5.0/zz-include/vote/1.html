投票

<stl:vote
	isDynamic="是否动态显示"
></stl:vote>

用于制作投票功能。

属性列表

下表显示所有属于此标签的属性列表。

属性 可能的取值 默认值 简介
voteName   第一个投票的名称 投票名称
width     显示宽度
itemWidth     投票项的宽度
columns   1 列数，默认显示一列
openWindowWidth   520 投票结果窗口宽度
openWindowHeight   280 投票结果窗口高度
titleStyle   font-weight:bold 投票标题显示样式
itemTextStyle     投票项文字显示样式
itemImageStyle     投票项图片显示样式
theme     主题样式
  Style1 主题样式1
  Style2   主题样式2
isDynamic     是否动态显示
  true   动态显示
  false 静态显示

子标签列表

下表显示此标签所有的子标签。

子标签 简介
stl:inputTemplate 投票提交模板，可以省略，省略将使用系统自带的样式显示投票

注意

<stl:vote> 可以不用定义子标签，系统将采用默认样式显示投票，当默认样式不满足需求时才需要使用 <stl:inputTemplate> 自定义投票提交样式。

  • 属性 voteName 为需要显示的投票名称，如下图所示：
  • 当使用 <stl:inputTemplate> 自定义投票提交样式时，属性 itemWidth、columns、titleStyle、itemTextStyle、itemImageStyle 及 theme 将失去作用。
  • 此标签可以放到任何模板中，系统将采用Ajax方式显示投票。

<stl:vote>标签对应的实体为{stl:vote}。
