用于显示投票提交项及其样式。

注意

  • stl:inputTemplate 内必须包含多个 <input type="radio"> 或 <input type="checkbox">，用于显示投票单选框或复选框。如果投票类型为单选，必须包含 <input type="radio">，如果投票类型为复选，必须包含 <input type="checkbox">
  • <input type="checkbox"> 或 <input type="radio">不用设置 id 或 name 等属性，系统将自动分配投票项 Id。
  • stl:inputTemplate 内必须包含 id="submit" 的 <button>、<input> 或 <a>标签，如 <input type="button" id="submit" value="提 交" />，用于提交投票。
  • stl:inputTemplate 内必须包含 id="results" 的 <button>、<input> 或 <a>标签，如 <input type="button" id="results" value="查 看" />，用于查看投票结果。
  • stl:inputTemplate 中除表单标签外可以添加任意 Html 标签，用于组织表单样式。
