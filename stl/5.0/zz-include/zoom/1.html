文字缩放 

<stl:zoom
	fontSize="缩放字体大小"
	isDynamic="是否动态显示"
	zoomId="页面HTML中缩放对象的ID属性"
></stl:zoom>

显示文字缩放链接，点击后缩放指定文字

属性列表

下表显示所有属于此标签的属性列表。
所有属于 HTML 标签 <a> 的成员均适用于 <stl:printer> 标签。 属于 HTML 标签 <a> 的成员未在下表列出。

属性 可能的取值 默认值 简介
zoomId     页面HTML中缩放对象的Id属性
fontSize     缩放字体大小
isDynamic     是否动态显示
  true   动态显示
  false 静态显示

注意

stl:zoom 标签将显示缩放链接，点击此链接后缩放指定文字。

<stl:zoom>标签对应的实体为{stl:zoom}。
